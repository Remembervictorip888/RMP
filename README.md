# RMP Product Cash Flow Projection

This project contains Excel VBA code for RMP (Risk Management Platform) product cash flow projection modeling.

## Overview

The RMP Product Cash Flow Projection is designed to forecast and analyze cash flows associated with risk management products. This tool helps financial analysts and risk managers to:

- Project future cash inflows and outflows
- Analyze various risk scenarios
- Evaluate product profitability over time
- Support decision-making processes

## Files

- `RMP_Excel_VBA_Code.bas` - Main VBA source code
- `RMP_Excel_VBA_V1.xlsm` - Excel workbook with embedded macros
- `Old_Version/` - Previous versions of the code for reference

## Features

- Dynamic cash flow calculations
- Scenario analysis capabilities
- Data visualization tools
- Automated reporting functions

## Getting Started

1. Open `RMP_Excel_VBA_V1.xlsm` in Microsoft Excel
2. Enable macros when prompted
3. Use the provided interface to input data and run projections

## Prerequisites

- Microsoft Excel 2016 or later
- Enabled macro security settings
